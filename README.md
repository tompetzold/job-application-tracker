## Startup

Docker Desktop manuell starten starten, danach:

```bash
docker compose up --build
```

Backend:

```text
http://localhost:8080/api/applications
```

Frontend:
```text
http://localhost:5173/
```

Swagger UI:
```text
http://localhost:8080/swagger-ui/index.html#/
```

Stoppen:

```bash
docker compose down
```

oder alternativ:
```text
control + c
```
